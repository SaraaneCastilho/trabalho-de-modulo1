Aspectos éticos e Legais da comunicação em sites

Ética é um conjunto de princípios morais que guiam os indivíduos ou um grupo da sociedade. Em se tratando da internet, ética no mundo digital é o que atua para manter dignidade, segurança, privacidade e outros valores no ambiente virtual, seguindo tanto os valores morais quanto as legislações a respeito do assunto. No Brasil, existem normas e regulamentações pensadas para manter alguns princípios de ética digital. São inúmeras as leis que visam proteger a dignidade humana, inclusive virtualmente, e um belo exemplo é o Marco Civil da Internet (Lei n. 12.965/2014), que garante o sigilo de dados pessoais, a privacidade de mensagens, as responsabilidades sobre o conteúdo e a neutralidade da rede.

Função e utilidade gestão de projetos

A gestão de projetos é a aplicação de ferramentas, técnicas e práticas para que um projeto seja bem-sucedido. Essa gestão proporciona ordem ao processo, otimizando-o para que o empreendimento seja realizado no menor tempo possível, aproveitando todos os recursos disponíveis.O modelo é dividido em fases de grupos de ações e tarefas que devem ser colocadas em prática ao decorrer de um projeto. Vejamos quais são essas fases: Iniciação. Planejamento. Execução. Monitoramento e controle. Encerramento.

GitHub e Git

O GitHub é um repositório online, que além de armazenar o seu projeto na nuvem, permite: • Trabalhar em equipe • Ramificações do projeto • Tornar o seu projeto público (onde outros usuários do GitHub podem opinar ou colaborar com o seu projeto) • É uma rede social de desenvolvedores • Seu GitHub pode ser o seu portfólio de projetos • Além de outras funcionalidades, como projetos privados, página web... O Git é um software que permite gerar repositórios e gerenciar versões, de uma forma mais inteligente, simples e com economia de espaço. Vantagens • Controle de histórico • Melhor gestão do espaço ocupado • É possível restaurar qualquer uma das versões anteriores • Organização • Segurança

Qual função do Docker?

Implantar aplicativos dentro de containers virtuais. A conteinerização permite que vários aplicativos funcionem em diferentes ambientes complexos. Usar container do Docker poupa aos usuários o incômodo de solucionar possíveis problemas de compatibilidade entre sistemas. Isso porque, com o Docker, um software é executado da mesma forma em todos os ambientes.
